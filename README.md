# CodeSage

AI-powered developer tool that understands entire codebases like a senior engineer.

## Vision
Turn raw source code into structured architectural knowledge that developers can query using natural language.

## Key Capabilities
- Repository analysis
- Function & module dependency graph
- Semantic code search
- Architecture explanations
- IDE integration

## Architecture
IDE Extension → Local AI Engine → Graph DB + Vector DB → AI Reasoning Engine

## Tech Stack
- Python
- FastAPI
- Tree-sitter
- Neo4j
- Qdrant
- PyTorch
- VSCode Extension (TypeScript)
- Docker

## V1 Features
- Repository indexing
- AST parsing
- Call graph generation
- Dependency graph
- Semantic search
- AI architecture Q&A
